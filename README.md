### 特色
1. 站點地圖化 - 直觀清楚的顯示GoBike各站點資訊
2. 一鍵導航 - 帶你走到您指定的GoBike站點位置
3. 地圖羅盤模式 - 讓你在地圖中不會迷失方向
4. 計時器(Beta) - 充分利用GoBike免費時間
